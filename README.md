# weather-precipitation-regression-databricks
# Weather Precipitation Regression on Databricks

## Overview

This project builds a simple Databricks regression pipeline using a free NOAA weather dataset from Databricks Marketplace.

The goal is to predict daily precipitation using station location and date-related features.
